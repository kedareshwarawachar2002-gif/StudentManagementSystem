# StudentManagementSystem
Build a console-based Student Management System in Core Java using Collections (ArrayList, HashMap). Implement CRUD operations, search and sort by name/grade, and serialize data to a file using Java I/O (ObjectOutputStream). Apply OOP principles throughout the design

Student Management System

Run the Project

```bash
mvn compile
mvn exec:java "-Dexec.mainClass=com.sms.app.Main"
```

Features

* CRUD operations
* Search by name
* Sort by name & grade
* File storage (students.dat)

Tech

Java, Maven, Collections, File I/O
